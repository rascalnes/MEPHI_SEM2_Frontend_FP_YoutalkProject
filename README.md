# Проект "Youtalk" — вёрстка блога о психологии

Этот проект представляет собой адаптивную вёрстку двух страниц сайта: главной страницы блога и страницы статьи.

## 🚀 Демо-проект

Посмотреть проект в действии можно здесь: [GitHub Pages](https://rascalnes.github.io/MEPHI_SEM2_Frontend_FP_YoutalkProject/)

## 📦 Как клонировать и запустить проект

1.  **Склонируйте репозиторий** на свой компьютер с помощью Git:
    ```bash
    git clone https://github.com/rascalnes/MEPHI_SEM2_Frontend_FP_YoutalkProject.git
	```

2. **Перейдите в папку проекта:**

bash```
cd MEPHI_SEM2_Frontend_FP_YoutalkProject
```

3. **Откройте файл blog.html в вашем браузере.** или article.html

## ✅ Просмотр

1.	Адаптивность: Откройте инструменты разработчика (F12) и меняйте размер экрана.
-	Десктоп (ширина > 992px): шапка с навигацией, сетка карточек из 3 колонок.
-	Планшет (ширина 768px - 992px): сетка карточек из 2 колонок, сайдбар статьи адаптируется.
-	Мобильный (ширина < 768px): бургер-меню, сетка из 1 колонки.
2.	Интерактивность:
-	Наведите курсор на любую кнопку или ссылку — курсор сменится на pointer, а элемент должен визуально изменяться (изменение цвета, затемнение).
-	Нажмите на бургер-меню (три полоски) в мобильной версии — откроется выезжающее меню. Нажмите на крестик, серую область или клавишу Esc — меню закроется.
3.	Семантика и доступность:
-	У всех изображений есть осмысленный атрибут alt.
-	Использованы теги <header>, <main>, <footer>, <article>, <aside>.
-	Кнопки и элементы фильтров имеют атрибуты aria-label для скринридеров.
4.	Качество кода:
-	CSS написан по методологии БЭМ (блоки вида card__title, card__image-wrapper).
-	В CSS используются псевдоклассы (:hover, :active) и медиа-запросы.
-	Иконки в формате SVG, картинки — в PNG.

🛠️ Стек технологий
-	HTML5 (Семантическая вёрстка, доступность)
-	CSS3 (Flexbox, Grid, медиа-запросы, БЭМ)
-	JavaScript (ES6, открытие/закрытие бургер-меню)
-	Git (Контроль версий)

## 💡 Особенности реализации проекта

-   **DRY-подход**: Шапка, подвал и мобильное меню вынесены в отдельные компоненты и подключаются через `fetch()`, что исключает дублирование кода на страницах `blog.html` и `article.html`.
-   **Кастомизация скролла**: На мобильных устройствах и планшетах у блока с тегами (фильтрами) появляется горизонтальный скролл для удобства навигации.
